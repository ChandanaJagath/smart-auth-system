# 🔐 Smart Auth System

Simple and secure authentication system using PHP, MySQL, and JavaScript.

---

## 🚀 Features

* Login & Register system
* Email verification
* Forgot password & reset
* User dashboard
* Admin panel (manage users & roles)

---

## 🔄 How it works

Register → Verify Email → Login → Dashboard
Forgot Password → Reset → Login

## ⚙️ How to run

1. Copy project to `htdocs`
2. Start XAMPP
3. Create database: `smart_auth`
4. Open in browser:

http://localhost/smart%20auth%20system/public
